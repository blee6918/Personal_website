# Brian (Minsoo) Lee - Personal Portfolio

Welcome to my personal portfolio website! This project showcases my work, skills, and interests as a Data Science student and Machine Learning Engineer.

## Overview

This is a static, single-page application built to be fast, responsive, and visually engaging. It features a modern "Bento Grid" layout, glassmorphism effects, and an interactive dock-style navigation.

**Live Demo:** [Insert Live Link Here if available]

## ‚ú® Features

-   **Bento Grid Layout**: A clean, modular design inspired by modern UI trends.
-   **Interactive Navigation**:
    -   **Floating Pill Navbar**: Easy access to Home, About, Work, and Contact sections.
    -   **Glass Dock**: A Mac-OS style dock for controlling the photo gallery.
-   **Dynamic Gallery**:
    -   Browse through different photo collections (Gallery, Travel, Hobbies, Dog).
    -   Click the dock icons to switch collections.
    -   Click the image to cycle through photos.
-   **Tech Stack Display**: Interactive icons linking to official documentation for Python, R, Pandas, TensorFlow, and more.
-   **Chatbot Interface**: A simulated chat interface in the Contact section to provide quick information about me (Email, LinkedIn, Resume, etc.).
-   **Responsive Design**: Optimized for both desktop and mobile viewing.
